Trigonometry Formulas and Proofs

Pythagorean Identity

sin²θ + cos²θ = 1

Fundamental identity relating sine and cosine functions

Sine Rule

a/sinA = b/sinB = c/sinC = 2R

Relates sides and angles of any triangle, where R is circumradius

Cosine Rule

a² = b² + c² - 2bc·cosA

Generalization of Pythagorean theorem for any triangle

General Trigonometric Solutions

sin(x) = sin(θ)
x = nπ + (-1)ⁿθ, n ∈ ℤ

cos(x) = cos(θ)
x = 2nπ ± θ, n ∈ ℤ

tan(x) = tan(θ)
x = nπ + θ, n ∈ ℤ

General solutions for trigonometric equations

Triple Angle Formulas

sin(3x) = 3sin(x) - 4sin³(x)

cos(3x) = 4cos³(x) - 3cos(x)

tan(3x) = (3tan(x) - tan³(x))/(1 - 3tan²(x))

Expansions for triple angles

Double Angle Formulas

sin(2θ) = 2sinθ·cosθ
cos(2θ) = cos²θ - sin²θ

Special Cases

sin(x) = 0
x = nπ, n ∈ ℤ

cos(x) = 0
x = (2n+1)π/2, n ∈ ℤ

sin(x) = 1
x = 2nπ + π/2, n ∈ ℤ

cos(x) = 1
x = 2nπ, n ∈ ℤ

Coordinate Geometry

Distance Between Two Points

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Distance between points (x₁, y₁) and (x₂, y₂)

Midpoint Formula

M = ((x₁ + x₂)/2, (y₁ + y₂)/2)

Coordinates of midpoint between two points

Slope of a Line

m = (y₂ - y₁)/(x₂ - x₁)

Slope between points (x₁, y₁) and (x₂, y₂)

Equation of a Line

y - y₁ = m(x - x₁)

Point-slope form of a line

Distance from Point to Line

d = |Ax₀ + By₀ + C| / √(A² + B²)

Distance from point (x₀, y₀) to line Ax + By + C = 0

Circle - Standard Form

(x - h)² + (y - k)² = r²

Center (h, k), radius r

Circle - General Form

x² + y² + 2gx + 2fy + c = 0

Center: (-g, -f), radius: √(g² + f² - c)

Distance Between Circle Centers

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Distance between centers of two circles

Section Formula (Internal)

P = ((mx₂ + nx₁)/(m+n), (my₂ + ny₁)/(m+n))

Point dividing line in ratio m:n internally

Section Formula (External)

P = ((mx₂ - nx₁)/(m-n), (my₂ - ny₁)/(m-n))

Point dividing line in ratio m:n externally

Area of Triangle

A = ½|x₁(y₂ - y₃) + x₂(y₃ - y₁) + x₃(y₁ - y₂)|

Area of triangle with vertices (x₁,y₁), (x₂,y₂), (x₃,y₃)

Permutation, Combination & Binomial Expansion

Permutation Formula

P(n, r) = n! / (n - r)!

Number of ways to arrange r objects from a total of n distinct objects.

Permutation with Repetition

nⁿ

When each object can be repeated, total number of arrangements is nⁿ.

Permutation of Similar Objects

n! / (p₁! · p₂! · p₃! · ...)

Used when some objects are identical, e.g., letters in “BALLOON”.

Circular Permutation

(n - 1)!

Number of ways to arrange n distinct objects in a circle.

Combination Formula

C(n, r) = n! / [r! (n - r)!]

Number of ways to choose r objects from n distinct objects (order doesn’t matter).

Binomial Theorem

(a + b)ⁿ = Σ [C(n, r) · aⁿ⁻ʳ · bʳ]

Expansion of powers of binomials using combinations.

General Term in Binomial Expansion

Tr+1 = C(n, r) · aⁿ⁻ʳ · bʳ

Formula to find the (r + 1)ᵗʰ term in the expansion of (a + b)ⁿ.

Example: Expansion of (x + 2)³

(x + 2)³ = C(3,0)x³ + C(3,1)x²(2) + C(3,2)x(2)² + C(3,3)(2)³

= 1·x³ + 3·x²·2 + 3·x·4 + 8

= x³ + 6x² + 12x + 8

Statistics Formulas

Mean (Average)

x̄ = (Σxᵢ) / n

The arithmetic mean is the sum of all values divided by the number of values.

Mode

Most frequent value in dataset

The value that appears most frequently in a data set.

Median

If n is odd: M = x(n+1)/2
If n is even: M = (xn/2 + x(n/2)+1) / 2

The middle value when data is arranged in order.

Variance

σ² = Σ(xᵢ - x̄)² / n

Average of squared deviations from the mean.

Standard Deviation

σ = √[Σ(xᵢ - x̄)² / n]

Square root of variance, measures data dispersion.

Probability

P(A) = n(A) / n(S)

Probability of event A is favorable outcomes divided by total outcomes.

Z-Score

z = (x - μ) / σ

Number of standard deviations a value is from the mean.

Standard Deviation Calculation Example

For dataset: 2, 4, 4, 4, 5, 5, 7, 9

1. Calculate mean: (2+4+4+4+5+5+7+9)/8 = 5

2. Find deviations: -3, -1, -1, -1, 0, 0, 2, 4

3. Square deviations: 9, 1, 1, 1, 0, 0, 4, 16

4. Sum of squares: 9+1+1+1+0+0+4+16 = 32

5. Variance: 32/8 = 4

6. Standard deviation: √4 = 2

Derivatives

Explore common derivatives and their applications:

Power Rule

d/dx(xⁿ) = n·xⁿ⁻¹

Derivative of x raised to any power n

Constant Rule

d/dx(c) = 0

Derivative of a constant is zero

Exponential Function

d/dx(eˣ) = eˣ

Derivative of exponential function is itself

Natural Logarithm

d/dx(ln x) = 1/x

Derivative of natural logarithm

Sine Function

d/dx(sin x) = cos x

Derivative of sine is cosine

Cosine Function

d/dx(cos x) = -sin x

Derivative of cosine is negative sine

Tangent Function

d/dx(tan x) = sec² x

Derivative of tangent is secant squared

Product Rule

d/dx(f·g) = f'·g + f·g'

Derivative of product of two functions

Quotient Rule

d/dx(f/g) = (f'·g - f·g')/g²

Derivative of quotient of two functions

Chain Rule

d/dx(f(g(x))) = f'(g(x))·g'(x)

Derivative of composite functions

Proof of Power Rule:

1. Start with definition of derivative: f'(x) = lim(h→0) [f(x+h) - f(x)]/h
2. For f(x) = xⁿ, we have f(x+h) = (x+h)ⁿ
3. Using binomial expansion: (x+h)ⁿ = xⁿ + n·xⁿ⁻¹·h + ... + hⁿ
4. Then [f(x+h) - f(x)]/h = [xⁿ + n·xⁿ⁻¹·h + ... - xⁿ]/h = n·xⁿ⁻¹ + ...
5. As h→0, all terms with h vanish, leaving n·xⁿ⁻¹
6. Therefore, d/dx(xⁿ) = n·xⁿ⁻¹
